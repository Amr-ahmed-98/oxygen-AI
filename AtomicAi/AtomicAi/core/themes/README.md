# Theme Packs

Complete theme packages that define all design tokens, typography, spacing, colors, shadows, and more for consistent styling across the entire system.

## Available Themes

1. **theme-minimal-clean** - Clean, minimalist theme with ample whitespace
2. **theme-saas-modern** - Modern SaaS theme with vibrant colors
3. **theme-agency-bold** - Bold, confident agency theme with high contrast
4. **theme-portfolio-editorial** - Editorial-style portfolio theme with elegant typography
5. **theme-ecommerce-premium** - Premium e-commerce theme optimized for conversions

## Theme Structure

Each theme includes:

### Foundations

- **colors**: Complete color system (primary, accent, neutral, semantic, background, text)
- **typography**: Font families, sizes, weights, line heights, letter spacing
- **spacing**: Spacing scale and values
- **borderRadius**: Border radius system (sharp/rounded/pill)
- **shadows**: Shadow system (subtle/modern/bold/soft/elevated)
- **motion**: Animation durations, easings, and reduced motion support

### Density Modes

- **compact**: Tighter spacing, smaller components
- **comfortable**: Balanced spacing (default)
- **airy**: Generous spacing, larger components

### Dark Mode

Complete dark mode color palette with automatic strategy support.

### RTL Support

Right-to-left language support configuration.

## Usage

Themes can be applied at build time or runtime to transform the entire component library's appearance.

## Benefits

- **Consistency**: All components styled consistently
- **Variety**: 5 complete themes ready to use
- **Customizable**: Easy to extend or create new themes
- **Production-ready**: Complete design token systems
- **Accessibility**: WCAG-compliant color contrast built-in
